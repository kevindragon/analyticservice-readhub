# analyticservice-readhub

analyticservice exercise.

    1. Fetch some data from readhub.cn
    2. Index these data
    3. Provide search functionality

## build

```
lein uberjar
```

## Usage

    $ java -jar analyticservice-readhub-0.1.0-standalone.jar port
